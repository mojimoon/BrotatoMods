# BrotatoMods

A collection of mods for the game [Brotato](https://store.steampowered.com/app/1942280/Brotato/).

## [Mojimoon-DoubleSidedUpgrades](Mojimoon-DoubleSidedUpgrades)

[Steam Workshop Link](https://steamcommunity.com/sharedfiles/filedetails/?id=3671945570)

Have you ever thought about the upgrades in Brotato being a bit too fixed? This mod allows you to have more control over your upgrades by introducing a double-sided upgrade system and an upgrades multiplier.

By default, 50% of the upgrades will be double-sided, which composes of a positive effect (200% of stat A) and a negative effect (-100% of stat B). Say, instead of choosing +20% attack speed at Tier 4, you can now choose a double-sided upgrade that gives you +40% attack speed but -12% movement speed.

For the following configurations, you need to subscribe to [dami-ModOptions](https://steamcommunity.com/sharedfiles/filedetails/?id=2944608034) to configure in-game: (this is not required if you are fine with the default settings)

This mod also allows you to modify the global, positive and negative upgrades multipliers, which can be set to any value you want. (If you are a masochist, you can even set the global multiplier to -100% to make the game more challenging!)

这个 mod 还允许修改全局、正面和负面升级的乘数，可以设置为你想要的任何值。（如果你是个M的话，不妨把全局乘数设置为-100%，让游戏更有挑战性！）
